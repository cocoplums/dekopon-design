## zh-CN

警告提示，展现需要关注的信息，适用于简短的警告提示。

## en-US

Warning prompts. Suitable for displaying short warning prompts in a way that attracts attention.

```js
import { Radio } from '@dekopon/design';

const App = () => {
  return (
      <>
        <Radio>
          1
        </Radio>
        <Radio>
          2
        </Radio>
      </>
  )
}

export default App
```
